# speech-playback

## Purpose
基于 Web Speech API 的英文语音朗读、嗓音选择与全局声音开关。

## Requirements

### Requirement: 英文语音朗读
系统 SHALL 使用浏览器原生 Web Speech API（`speechSynthesis`）朗读英文文本，并使用适合幼儿的、接近自然的语速与音高。

#### Scenario: 朗读单词
- **WHEN** 应用请求朗读一个英文单词或字母
- **THEN** 系统以英文语音、适合幼儿的语速朗读该文本

#### Scenario: 连续朗读
- **WHEN** 在前一段语音尚未结束时请求新的朗读
- **THEN** 系统取消正在进行的朗读并立即开始新的朗读，避免语音排队堆叠

### Requirement: 嗓音质量优选与选择
系统 SHALL 自动从可用英文嗓音中优选音质自然的嗓音、避开已知机械/趣味嗓音，并提供嗓音选择控件供用户切换；所选嗓音 SHALL 被持久化记忆，控件 SHALL 以中文标注口音与性别。

#### Scenario: 自动优选嗓音
- **WHEN** 应用加载可用嗓音列表
- **THEN** 系统按音质评分优先选中自然嗓音，并排除已知低质量嗓音

#### Scenario: 用户切换嗓音
- **WHEN** 用户在嗓音选择控件中选择另一个嗓音
- **THEN** 系统改用该嗓音、试听一次，并记住该选择供下次使用

### Requirement: 全局声音开关
系统 SHALL 提供一个全局声音开关控件，孩子或家长可随时开启或关闭所有语音，且关闭状态下视觉反馈仍正常工作。

#### Scenario: 切换声音开关
- **WHEN** 用户点击声音开关按钮
- **THEN** 系统在开启与关闭状态间切换，并更新按钮图标以反映当前状态

#### Scenario: 关闭状态下不发声
- **WHEN** 声音处于关闭状态且发生任意需要发声的交互
- **THEN** 系统不播放任何语音

### Requirement: 语音不可用时的降级
系统 SHALL 在浏览器不支持 `speechSynthesis` 时优雅降级，保证视觉内容与交互仍可正常使用。

#### Scenario: 浏览器不支持语音
- **WHEN** 运行环境不支持 Web Speech API
- **THEN** 系统跳过语音播放，不报错，所有图片、卡片与游戏交互继续可用
