## Why

幼儿园小朋友（3-6岁）识字量少，传统英语学习材料以文字为主，不适合他们。需要一个以图片、声音、动画为主，几乎不依赖阅读能力的网页，让孩子通过点一点、听一听、玩一玩的方式启蒙英语。沿用仓库里已有的"椰子"系列可爱风格，保持品牌一致性。

## What Changes

- 新增一个纯前端网页项目 `CoconutEnglish`（零依赖，浏览器打开即用）。
- 单词主题学习：按动物、水果、颜色、数字等主题分组的大图标单词卡，点击即用浏览器语音合成（Web Speech API）朗读英文。
- 字母 ABC 启蒙：A-Z 字母卡，每个字母配代表单词和图标，点击发音。
- 听音找图小游戏：播放一个英文单词，孩子从几张图片中点选正确的，答对有鼓励动画和音效，答错温柔提示、无惩罚。
- 用可爱卡通风格的矢量插画（内联 SVG，非 emoji）作为图标，保证高清、可缩放、风格统一；图标结构可替换，便于后续升级为更精致的插画素材。
- 护眼用眼提醒：每使用 20 分钟弹出休息提示弹框，引导孩子和家长休息眼睛。
- 全局声音开关、大按钮、护眼明亮配色，适配触屏点击。

## Capabilities

### New Capabilities
- `word-learning`: 主题单词卡的展示与点击发音学习。
- `alphabet-learning`: A-Z 字母启蒙卡与发音。
- `listening-game`: 听音找图的互动小游戏，含计分与正负反馈。
- `speech-playback`: 基于 Web Speech API 的英文语音朗读与全局声音开关。
- `break-reminder`: 每 20 分钟弹出休息提醒，保护幼儿视力。

## Impact

- 新增目录 `CoconutEnglish/`，包含 `index.html`、`style.css`、`app.js`、`data.js`。
- 无后端、无构建步骤、无第三方依赖。
- 图标采用内联 SVG 卡通插画（可放大不失真），不依赖外部图片；后续可替换为位图素材。
- 依赖浏览器原生 Web Speech API（`speechSynthesis`），需在现代浏览器中运行；不可用时优雅降级（仍可看图，无声音）。
